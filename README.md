# 🏥 AI-Powered Conversational Healthcare Analytics Platform

An AI-powered Healthcare Analytics Dashboard built using **Python, Streamlit, Plotly, Pandas, and Google Gemini AI**. The platform enables users to analyze patient readmission data using interactive visualizations, natural language queries, voice input, and AI-generated insights.

# ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/Zulekhamshaikh/AI-Powered-Conversational-Healthcare-Analytics-Platform.git
```

### Navigate to the Project

```bash
cd AI-Powered-Conversational-Healthcare-Analytics-Platform
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Configure Environment Variables

Create a `.env` file in the project root:

```env
GEMINI_API_KEY=YOUR_API_KEY
```

### Run the Application

```bash
streamlit run app.py
```
